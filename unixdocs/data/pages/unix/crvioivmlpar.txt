====== Create a VIO lpar ======


  - Access the IVM Web UI and click on create lpar 
    - {{:unix:01clickcrpartition.png|}}
  - Enter an lpar name
    - {{:unix:02enterlparname.png|}}
  - Enter memory parameters
    - {{:unix:03entermemory.png|}}
  - Enter CPU parameters
    - {{:unix:04enter_processor.png|}}
  - Enter network configuration info
    - {{:unix:05selectnetwork.png|}}
  - Enter storage type
    - {{:unix:06storagetype.png|}}
  - Select the disks
    - {{:unix:07selectthedisk.png|}}
  - Select Optical drive
    - {{:unix:08opticaldrive.png|}}
  - Click finish on the sumary page
    - {{:unix:09summary.png|}}
  - And here is the lpar
    - {{:unix:11lparcreated.png|}}
