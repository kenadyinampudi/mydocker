====== Powerpath Notes ======

  * Configure Powerpath <code> powermt config</code>
  * Display Powerpath <code>powermt display</code>
  * Display all Powerpath devices <code>powermt display dev=all</code>
  * Set adapter offline <code>powermt set adapter <no> offline</code>
  * Set adapter to standby <code>powermt set adapter <no> standby</code>
