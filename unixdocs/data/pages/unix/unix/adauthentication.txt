====== AIX authentication using MS AD LDAP. ======

There are two common ways,

  * Kerberose 5 ( Users need to be created locally - password authentication is done against Active Directory )
  * LDAP ( Enable 'Identity Management for UNIX' and configure LDAP on AIX  - This allows , password and other unix attributes to be stored in AD )
