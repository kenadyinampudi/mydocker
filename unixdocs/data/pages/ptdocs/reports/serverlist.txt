===== Server List =====
This list is generated by first running 
  machinery inspect <servername>
\\ 
This uses ssh keys to log into the server and gather data.  This is stored
locally somehow.  The next step is to export this as HTML so the Wiki can read
it.\\ 
  machinery export-html --force -d /mnt/data/pages/reports/<servername> <servername>

Finally, do some cleanup work to make sure the output works with DokuWiki.\\

  mv <servername>/index.html <servername>/index.txt
  chown -R 33:33 <servername>
