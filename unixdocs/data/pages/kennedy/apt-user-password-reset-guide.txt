====== Ansible Provisioning tool(APT) user password reset guide ======
  - Ansible Web URL[[https://pcdcawxlxap01:8043/#/home]]
  - Log in using your ID.
  - After you log in go to Templates page,
  - The template page has a template to reset your password, At the right-hand corner, you have a launch button with which you can launch the template. 
  - Fill in the your new password and select the server you want to reset, you can type "all" if you want to reset your password on all the servers.
  - After that you will be taken to the output page where you would see what happened, look for the successful output.
